# TYPE DOCUMENTATION ## This document provides a definition of all types mentioned in API Documentation in this document. To connect to AWS Dev Instance - use the DEVNET Environment To connect to localhost - use the LOCAL Environment #### Type Post ``` typescript type Story = { "_id": Sid, "user_id": Uid, "description": string, "assets": Asset[], "likes": Uid[], "shares": Share[], "location": Location "comments": Comment[], "timestamp": "2023-06-10T19:28:14.984Z", "__v": 0 } type Sid = string; type Uid = string; type Asset = {}; type Share = {}; type Location = {}; type Comment = {}; timestamp = UTCDatetimeStamp = string; ``` #### Type User ``` typescript type User = { "_id": Uid "username": string, "fullname": string, "email": string, "bio": "", "profile_image_uri": string, "stories": Sid[], "followers": [], "following": [], "notifications": Nid[], "source": Source, "source_id": SourceID, "notification_tokens": NotificationToken[], "saved_posts": [], "timestamp": "2023-06-19T11:12:23.819Z", "__v": number } ``` #### Type Suggestion ``` typescript type suggestion_id = string; ```
